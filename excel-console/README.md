# Excel-console

## About

<p>Excel console application.</p>

## Pre-setup

Run the following command in terminal to install the project dependencies:
```
npm i
```

## Commands

Use following command in terminal for build and run script.
```
npm run dev
```

Use following command in terminal for only build.
```
npm run build
```

Use following command in terminal for and run script.
```
npm start
```
